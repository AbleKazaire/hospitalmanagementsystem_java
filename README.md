 Hospital Management System

A Java-based Hospital Management System designed to streamline hospital operations, manage patient records digitally, and improve healthcare service delivery through a structured and efficient database system.


 Project Overview

The Hospital Management System (HMS) is a desktop-based application developed using Java for the front-end and application logic, with MySQL as the relational database.

The system is built to:

* Digitize patient registration and records
* Improve data organization within hospitals
* Reduce paperwork and manual errors
* Enhance efficiency in managing hospital workflows


 **Key Features**
